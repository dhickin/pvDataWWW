pvData: EPICS PVData

Description

This project has the Java implementation of PVData.

Getting Started

The documentation is available via JavaDOC For the latest version ask for Description in doc/index.html

pvData is being developed as an eclipse project. Each release is available as a zip file that contains the source and pre-built Java class libraries.the easiest way to try the pvData is to import the zip file into an eclipse project.
