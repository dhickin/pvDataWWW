javaIOC: EPICS PVData Input/Output Controller

2009.10.05

Description

JavaIOC is a JAVA implementation of an EPICS IOC. It has many similarities to a Version 3 ( the 3.13 and 3.14 releases of EPICS base) but extends the data types to support structures and arrays.

Getting Started

The documentation is available via JavaDOC For the latest version ask for Description in doc/index.html

JavaIOC is being developed as an eclipse project. For now release are only available via the sourceforge CVS repository.

This project depends on projects pvData and CAJv4. It also requires the SWT (Standard Widget Toolkit) which does come with eclipse.
