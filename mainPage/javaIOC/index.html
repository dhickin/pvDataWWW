javaIOC: EPICS PVData Input/Output Controller

Description

JavaIOC is a JAVA implementation of an EPICS IOC. It has many similarities to a Version 3 ( the 3.13 and 3.14 releases of EPICS base) but extends the data types to support structures and arrays.

Getting Started

The documentation is available via JavaDOC For the latest version ask for Description in doc/index.html

JavaIOC is being developed as an eclipse project. Each release is available as a zip file that contains the source and pre-built Java class libraries.the easiest way to try the javaIOC is to import the zip file into an eclipse project. In the future better packaging will be provided.

Releases

For now release are only available via the sourceforge CVS repository.
