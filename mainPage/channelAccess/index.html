CAJv4: EPICS PVData Channel Access

Description

Channel access is the component of EPICS that makes process variable data available via the network. CAJv4 is a Java implementation of channel access which supports PVData.

Getting Started

The documentation is available via JavaDOC For the latest version ask for Description in doc/index.html

CAJv4 is being developed as an eclipse project. Each release is available as a zip file that contains the source and pre-built Java class libraries.the easiest way to try the CAJv4 is to import the zip file into an eclipse project.

It is also possible to the CAJv4 without eclipse. The unzip command creates a directory named CAJv4 that has the following structure:

CAJv4/
    bin    -  The prebuilt Java classes
    doc    -  The javaDoc. Start with doc/index.html
    src    -  The Java sources
    xml    -  Database Definition xml files for support provided by CAJv4

Add the CAJv4/bin directory to your CLASSPATH. However you must also get the support for the Standard Widget Toolkit (SWT) that is provided by eclipse. It is much easier to just make the CAJv4 an eclipse project.
